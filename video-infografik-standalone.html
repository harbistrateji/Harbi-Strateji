Dışa Aktarım & Kullanım Kılavuzu

Bu sayfadaki tüm veriler, harici video düzenleme araçlarında kullanılmak üzere yapılandırılmıştır:

1. WebM Video: Ana uygulamadan (/video-infografik) her sahneyi 9:16, 16:9 veya 1:1 formatında WebM olarak indirebilirsiniz.

2. JSON Veri: video-infografik-data.json dosyasını After Effects (Bodymovin/Lottie), Remotion, FFmpeg veya Canva/Figma'da kullanabilirsiniz.

3. Canva/Figma: Her sahnenin veri yapısını kopyalayıp şablon tasarımlarınıza aktarabilirsiniz.

4. Sosyal Medya: YouTube (16:9), Instagram Reels/TikTok (9:16), Instagram Post (1:1) formatları desteklenir.

Örnek JSON yapısı:

{
  "id": "guc_formulu",
  "title": "Gücün Algoritması",
  "category": "Formül",
  "duration": 10,
  "content": {
    "type": "formula",
    "formula": "G = T × S × E × L",
    "variables": [
      { "symbol": "T", "name": "Teknoloji", "value": 7 },
      { "symbol": "S", "name": "Sistem Entegrasyonu", "value": 6 }
    ],
    "result": "G = 1.680"
  }
}
